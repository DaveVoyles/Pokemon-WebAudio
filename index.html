Pokemon Finder

Gotta say 'em all!

This web app takes advantage of Web Components and Polymer to enable new HTML features in the browser.

In this particular case, we are using the x-pokemon web component to pull the images from a database, as well as the voice-elements web component to speak the name of the pokemon we entered.

Enter the name of a Pokemon, watch it display on screen, then press the Speak button to hear the pronounciation.

Not sure of a name? You can find a bunch of them here: PokemonDB
